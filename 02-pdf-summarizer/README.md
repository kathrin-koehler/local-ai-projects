# 02 · PDF & Text Summarizer

Summarizes any PDF or text file locally using Ollama and llama3.2.
No cloud, no API key, no cost.

## Usage

1. Place your file in this folder
2. Set the filename in `summarizer.py`
3. Run:
```bash
   python3 summarizer.py
```

## Dependencies
```bash
pip3 install ollama pypdf2
```
