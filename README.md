# Local AI Projects

Personal learning project: building AI tools and agents locally with Ollama and Python.
No cloud, no costs, no data leaving my machine.

## Setup

- **Hardware:** MacBook Air M2
- **LLM Runtime:** Ollama (local)
- **IDE:** VS Code + Continue Extension
- **Language:** Python
- **Cost:** 0 €/month

## Projects

### ✅ 01 · Local Setup & Copilot
Installed Ollama, loaded llama3.2, connected to VS Code via Continue Extension.
Result: fully local AI coding assistant.

### 🔜 02 · PDF & Text Summarizer
Python script to summarize any PDF or text file locally.

### 🔜 03 · AI Agent
An agent that decides which steps to take: answer directly, read a file, or process data. Basically teaching a computer to think before it acts (... still waiting for humans to do the same).

### 🔜 04 · RAG System
Making own documents searchable with embeddings and a local vector database.

## Why local?

Running LLMs locally means full privacy, no API costs, and a deeper understanding
of how these models actually work, beyond just calling an API.
Swabian-approved: if it can be done for free, it will be done for free.
